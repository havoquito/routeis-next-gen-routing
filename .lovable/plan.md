

# Routeis.com — Landing Page de Alta Conversão

## Visão Geral
Landing page developer-first para o Routeis, um framework de roteamento JavaScript. Design dark mode com acentos em roxo elétrico/verde neon, glassmorphism e estética de velocidade.

---

## Estrutura & Seções

### 1. Hero Section
- Título bold: **"The Smartest Route for your Modern Stack."**
- Subtítulo descritivo sobre performance e universalidade
- Dois CTAs: **"Get Started"** (botão primário com gradiente roxo/neon) e **"View on GitHub"** (botão outline)
- Bloco de terminal estilizado com `npm install routeis` e animação de digitação
- Partículas ou linhas de fluxo sutis no background para transmitir velocidade

### 2. Recursos Principais (3 Colunas)
Cards com efeito glassmorphism (fundo semi-transparente com blur):
- **Zero Config** — ícone de pasta, configuração automática baseada em diretórios
- **Type Safe** — ícone de TypeScript/shield, integração nativa com TS
- **Edge Ready** — ícone de raio/cloud, otimizado para Cloudflare Workers e Vercel Edge

### 3. Code Showcase
- Editor de código estilizado (visual similar a VS Code) com syntax highlighting
- Exemplo de criação de rota simples com `import { route } from 'routeis'`
- Barra superior com dots coloridos e nome do arquivo
- Fundo escuro com bordas glassmorphism

### 4. Benchmarks / Social Proof
- Gráfico de barras comparativo mostrando Routeis **2x mais rápido** que concorrentes
- Barras animadas com labels (Routeis vs Express Router vs React Router)
- Métricas destacadas: requests/sec, bundle size, tempo de cold start

### 5. Footer
- Links organizados em colunas: Docs, Community (Discord), GitHub, Twitter/X
- Logo Routeis pequeno
- Copyright

---

## Estilo Visual
- **Paleta:** fundo escuro (#0a0a0f), cards com glassmorphism, acentos em roxo elétrico (#8B5CF6) e verde neon (#22D3EE)
- **Tipografia:** Sans-serif moderna (Inter/system font), títulos bold
- **Efeitos:** glassmorphism nos cards, gradientes sutis, glow effects nos CTAs, animações de entrada ao scroll
- **Responsivo:** layout adaptável para mobile com stack vertical
